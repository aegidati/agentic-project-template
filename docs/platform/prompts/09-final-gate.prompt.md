# Prompt 09 - Final Gate

## Context
Run final readiness gate before normal feature delivery starts.

## Prompt To Run In Copilot Chat
Run a final gate and declare READY or NOT READY. Validate structure, documentation consistency, canonical starter paths unchanged (app/backend, app/web, app/client, app/contracts, app/infra, app/composition), and post-install validation outcomes.

## Done Criteria
1. Final gate result is declared as READY or NOT READY.
2. Blocking issues are listed if NOT READY.
3. Immediate next action is proposed.
